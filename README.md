# Utthan Singh Roy | Personal Portfolio

A premium, production-quality personal portfolio designed to showcase engineering excellence in Artificial Intelligence, Quantitative Finance, and Power Systems. 

Built with a focus on minimalism, modern typography, and sophisticated interactions, this portfolio intentionally avoids generic templates (like Bootstrap or Tailwind) to deliver a bespoke, Apple/Linear-inspired web experience.

## 🛠️ Architecture & Tech Stack
This project is engineered from scratch without reliance on heavy UI frameworks:
* **Semantic HTML5:** Clean, accessible markup.
* **Modern CSS3:** Custom variables, CSS Grid/Flexbox, and performant cubic-bezier transitions.
* **Vanilla JavaScript:** Lightweight intersection observers for scroll reveals and sticky navigation.
* **Typography:** Space Grotesk (Headings) & Inter (Body).
* **Icons:** Phosphor Icons (Minimalist web font).

## 🧠 Featured Domains
* **Artificial Intelligence:** Scalable ML pipelines and predictive modeling.
* **Quantitative Finance:** Mathematical frameworks and market regime forecasting.
* **Electrical Engineering:** Resilient grid infrastructures and digital twins.

## ⚙️ Local Development
To view or modify this project locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/ExcelsiorSR/portfolio.git](https://github.com/ExcelsiorSR/portfolio.git)
   ```

Navigate to the project directory:

```bash
cd portfolio
```
Open index.html in your preferred web browser, or use a local development server (like VS Code Live Server) to preview changes.
